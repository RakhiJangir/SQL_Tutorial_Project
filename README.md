What is SQL and Database? 

Answer: 
SQL stands for structured query language which is used to communicate with data in relational database.
A database is a collection of structured data. Structured data is organized in ralational tables with rows and columns.

SQL Data Types?

Answer:
SQL data types are generally categorized into a few main types, which tells what kind of data a column can hold such as
Integer data type, String/Text data type, Date and time data type, Boolean data type, Binary or large object data type, specialized data type
